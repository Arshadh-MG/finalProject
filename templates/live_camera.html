How Live Monitoring Works
  1. Add a camera by providing a Camera ID and selecting a source.
  2. If "Laptop Webcam" is chosen, the system uses your default camera.
  3. If "Forest Camera" is chosen, a pre‑configured RTSP URL is used (you can change it in the code).
  4. If "Custom RTSP URL" is chosen, enter the URL manually.
  5. The system continuously processes frames in real‑time.
  6. YOLOv8 detects animals in each frame.
  7. SlowFast Network analyzes motion patterns using a sliding window buffer.
  8. Injury indicators detected: limping, leg dragging, sudden collapse, prolonged immobility, distress patterns.
  9. Alerts are generated immediately when injury probability exceeds 75%.
  10. Duplicate alerts are prevented within 5 minutes per camera zone.
